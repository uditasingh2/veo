============================================================
🎬  Pixelbin 24/7 Automated Video Generator & API
============================================================
1. Purpose: Automates AI video generation using guest sessions.
2. Limit Bypass: Automatically clears LocalStorage to reset credits.
3. 24/7 Cloud: Runs on Render's Free tier via Express server.
4. Keep-Alive: Uses /ping endpoint with external pinger.
5. Setup: Connect GitHub repo (uditasingh2/veo) to Render.

------------------------------------------------------------
🚀 HOW TO USE AS AN API (Public)
------------------------------------------------------------
Trigger a generation via URL:
https://pixelbin-gen.onrender.com/generate?prompt=YOUR_PROMPT

Step 1: Click the URL above. It will return "Generation started!"
Step 2: Wait 2 minutes.
Step 3: Check https://pixelbin-gen.onrender.com/status for your video link.

Example:
https://pixelbin-gen.onrender.com/generate?prompt=A+cute+cat+in+space
